# ISH Sales Accelerator — D365 Frontend

Frontend-only implementation of the India Sweet House Sales Hub / Lead Accelerator.

## Stack

- Next.js 16 — App Router, RSC + client components
- React 19 + TypeScript
- Tailwind CSS v4 — custom tokens in `src/styles/tokens.css`
- shadcn/ui — Base UI + Radix primitives
- Lucide, Sonner, next-themes

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000

Original prototype: `reference/ISH_SalesAccelerator_D365.jsx`
